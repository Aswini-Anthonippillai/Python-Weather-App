# 🌦 Weather App (Python)

This is a simple weather application built using Python and Tkinter.

## Features
- Get real-time weather data
- Enter any city name
- Displays temperature and condition

## Technologies Used
- Python
- Tkinter
- OpenWeather API

## How to Run
1. Install requests:
   pip install requests
2. Add your API key
3. Run:
   python weather_ui.py
